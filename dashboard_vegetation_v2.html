Method Notes

Land cover classification derived from NSW State Vegetation Type Map (2023) supplemented with Sentinel-2 imagery analysis for current conditions.

Fractional cover from Geoscience Australia's Landsat-derived product (v3.1), providing annual estimates of photosynthetic vegetation, non-photosynthetic vegetation, and bare soil at 30m resolution.

Vegetation greenness (NDVI) derived from MODIS 16-day composites (250m resolution), showing annual mean, minimum, and maximum values across 2000–2024.

Regional benchmarks compare this property to locations within 50 km with similar land use. The regional median is shown as a reference line; the property value is shown as a dot.

Tree cover change combines Global Forest Change dataset (Hansen/UMD) with NSW SLATS (Statewide Landcover and Trees Study) for validation.

All spatial data referenced to GDA2020 / MGA Zone 55. Analysis period: 1990–2024 where data available.
